# Subhash Portfolio

A content-driven, statically-generated site showcasing Subhash Mahimaluri's resume and interview questions.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

For a detailed technical overview, refer to the [Architecture Documentation](architecture.md).
