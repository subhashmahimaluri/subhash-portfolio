# Subhash Portfolio

A Next.js application to display Subhash Mahimaluri's resume and portfolio.

For a detailed technical overview, please refer to the [Architecture Documentation](architecture.md).

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
